## Project 2 - HTML Foundation Page

### Webpage Theme
My webpage is about the Philadelphia Eagles and includes information about the team, their history, and the 2024 roster.

### What I Learned
While creating this page, I learned how to structure an HTML document using headings, paragraphs, images, tables, and navigation links. I also learned how to use attributes like src, href, and style.

### Challenges I Faced
One challenge I faced was getting my images to display correctly due to file path issues. I also had to fix validation errors like missing tags and character encoding problems.



# GreenTech Solutions Website

## Project Description
This project is a responsive website created for GreenTech Solutions, a company focused on environmental sustainability and eco-friendly technology. The website was designed to inform users about the company’s products and services.

## Setup and Running Instructions
1. Download or clone the project files.
2. Open the project folder in Visual Studio Code.
3. Make sure all image files are placed inside the `images` folder.
4. Open the `index.html` file in a web browser.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Visual Studio Code
